## Description
This repository contains a collection of useful cheat sheets that I use in my
day to day work. It contains 4 cheat sheets:
* Vim
* Tmux
* Markdown
* Git

You can directly read them or use the `cheat-reader.sh`.

```bash
$ cheatread.sh <cheatsheet>
```
